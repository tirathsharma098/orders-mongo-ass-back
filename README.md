# Techno orders app

## Requirements

- [Node.js](https://nodejs.org/) (v20 or higher)
- [MongoDB](https://www.mongodb.com/) (v4 or higher)

## Getting Started

### Installation

1. Clone the repository:

   ```bash
   git clone ---

2. npm install

3. Run the seed file:
    
    ```bash
    node src/seed/insertOrder.js

3. Start the application:
    
    ```bash
    node index.js

4. Api endpoint:
    ```bash
    /api/v1/order/get-list

    /api/v1/order/total-sales
